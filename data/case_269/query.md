## Context
This week at SuperBytes, a manufacturing issue has resulted in a number of items needing to be recalled from the shelves.

## Requirements

- Input the data
- Identify which recalled items are located in each store
- Create a field which shows how much money will be lost for each item
- Calculate how much money will be lost for each category, and round it to 2 decimal places
- Output this first dataset
- Find the total money lost for each store, rounding to 2 decimal places
- Create a new field that identifies stores as High Priority (lost £5000 or more) or Low Priority (below £5000)
- Output this second dataset

## Output

- output_01.csv
  - 2 fields:
    - Category
    - Total Price Rounded

- output_02.csv
  - 3 fields:
    - Store
    - Total Price Rounded
    - Issue Level
